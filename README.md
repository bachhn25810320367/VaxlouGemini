## ⚡ Các Tính Năng Đột Phá — Thiết Kế "WOW"

### 📁 1. Bookmark Bar & Dynamic Folders (Quản lý File & Chat Phân Cấp)
*   **Segmented Capsule Tabs (Arc Style)**: Thanh Bookmark Bar được thiết kế theo phong cách Arc Browser tối giản với các tab viên nhộng bo tròn mềm mại, chuyển đổi nảy đàn hồi cực sống động.
*   **Thư mục con lồng nhau (Subfolders)**: Kéo thả các cuộc hội thoại Gemini native thả trực tiếp vào các thư mục hoặc thư mục con. Tự động ẩn các cuộc chat đã gán khỏi thanh lịch sử native để giữ giao diện ngăn nắp.
*   **Đóng chéo thông minh (Cross-Closing Menus)**: Cơ chế đóng chéo tự động thông minh giữa các thư mục Bookmark Bar, Bento Menu (`⠿`), và trình gán nhanh (`+ Thêm chat vào file`). Khi mở bất kỳ menu tác vụ nào, các menu khác sẽ tự động đóng lại và dọn sạch trạng thái `.active` để không bao giờ xảy ra xung đột thị giác.
*   **Dynamic Breadcrumbs Swapper**: Khi bạn đang ở trong một cuộc chat đã được phân loại vào thư mục, Logo Gemini ở góc trái Bookmark Bar sẽ tự động biến đổi thành **đường dẫn phân cấp động tuyệt đẹp** (`Thư mục > Thư mục con > Tiêu đề Cuộc chat`) với tông màu gradient phản quang cao cấp, tự động khôi phục logo chuẩn Google khi chuyển sang chat tự do.

### 📝 2. Rich Notes Station (Trạm Ghi Chú Đa Năng Thời Gian Thực)
*   **Auto-Save & Status Indicator**: Ghi chú tự động lưu ngầm thông minh sau mỗi 700ms với bộ xoay indicator trạng thái đồng bộ hóa mượt mà, trực quan.
*   **Phạm vi lưu trữ kép (Scoped Storage)**: Cho phép lưu trữ Ghi chú Toàn cục (Global) hoặc Ghi chú riêng biệt theo từng Cuộc hội thoại (Chat-Scoped) chỉ bằng một nút gạt chuyển đổi nhanh.
*   **Paste & Drag-Drop Hình Ảnh**: Sao chép hoặc kéo thả ảnh trực tiếp vào trình biên tập rich text, ảnh được mã hóa và chèn inline mượt mà.
*   **Bộ phím tắt soạn thảo nhanh**: In đậm (`Bold`), In nghiêng (`Italic`), Gạch chân (`Underline`), Gạch ngang (`Strikethrough`) trực tiếp bằng phím tắt chuẩn.

### 🧭 3. Interactive Outline Navigator (Dàn Ý Tương Tác Sắc Nét)
*   **Trích xuất cấu trúc AI tự động**: Tự động phân tích toàn bộ câu hỏi của bạn và tiêu đề (H1-H4) của Gemini để tạo ra một bản sơ đồ dàn ý phân cấp rõ ràng.
*   **Bản Đồ Cuộn Cực Nhạy (Smooth Scroll)**: Nhấp vào bất kỳ mục nào trong Dàn ý để cuộn mượt mà đến đúng khung bong bóng chat trên màn hình, kèm hiệu ứng **viền bao quanh phát sáng (Chat Bubble Glow)** để định hướng mắt nhìn ngay lập tức.
*   **Bộ lọc thông minh**: Lọc nhanh chỉ xem câu hỏi của bạn (`Prompts`) hoặc chỉ xem câu trả lời của AI (`Responses`).

### ⚡ 4. AI Context Condenser (Bộ Cô Đọng Ngữ Cảnh Tối Tân)
*   **Giải phóng bộ nhớ token**: Khi cuộc trò chuyện quá dài dẫn đến lag, click `⚡ Bắt đầu cô đọng với AI` để tóm tắt toàn bộ lịch sử trò chuyện (nhiệm vụ đang làm, việc đã hoàn thành, code quan trọng, lỗi hiện tại, bước tiếp theo) thành dạng Markdown siêu gọn.
*   **One-Click Continue**: Nhấp nút `🚀 Mở chat mới & Tiếp tục công việc` để tự động mở một tab chat Gemini mới tinh, chèn toàn bộ bản tóm tắt ngữ cảnh đã cô đọng vào khung prompt, giúp bạn tiếp tục công việc trên một môi trường siêu nhẹ mà không bị mất dấu vết.

### 🔑 5. API Key Security Shield & Active Model Controller
*   **Bảo mật khóa vĩnh viễn**: Khung nhập API Key luôn được khóa chặt ở dạng `type="password"` bảo mật tuyệt đối, ẩn vĩnh viễn sau khi lưu để ngăn ngừa rò rỉ khóa khi chia sẻ màn hình.
*   **Accordion Technical Gọn Gàng**: Gom toàn bộ phần cấu hình technical (Khóa Gemini, khóa Groq, model active, quản lý API) vào accordion `⚙️ Cấu hình AI & API nâng cao` và mặc định thu gọn (collapsed) để giữ giao diện Cài đặt luôn tinh tế như một sản phẩm polished consumer.
*   **Model Router & Overload Handler**: Tự động kiểm tra và chuyển hướng gọi API Gemini/Groq tương ứng theo model bạn chọn. Khi Gemini native bị từ chối/quá tải, hệ thống hiện card cứu hộ cứu nạn ngay lập tức để chuyển tiếp tự động sang chat mới thông qua API key dự phòng.

## 🛠️ Hướng Dẫn Cài Đặt (Developer Mode)

Bạn có thể cài đặt thủ công tiện ích vào Google Chrome cực kỳ dễ dàng theo các bước sau:

1.  **Tải mã nguồn về máy**:
    *   Tải thư mục chứa mã nguồn extension này về máy hoặc clone trực tiếp bằng Git:
    ```bash
    git clone https://github.com/yourusername/vaxlou-v2.git
    ```
2.  **Mở trang quản lý Tiện ích của Chrome**:
    *   Mở trình duyệt Chrome và truy cập đường dẫn: **`chrome://extensions/`**.
3.  **Kích hoạt Chế độ nhà phát triển (Developer Mode)**:
    *   Gạt nút **"Developer mode" (Chế độ nhà phát triển)** ở góc trên cùng bên phải sang trạng thái **BẬT (ON)**.
4.  **Nạp tiện ích đã giải nén**:
    *   Click vào nút **"Load unpacked" (Tiện ích đã giải nén)** ở góc trên bên trái.
    *   Chọn thư mục chứa mã nguồn extension (thư mục chứa tệp `manifest.json`).
5.  **Hoàn tất**:
    *   Truy cập trang [Google Gemini](https://gemini.google.com/) chính thức, nhấn **Ctrl+F5** để làm sạch cache trình duyệt và thưởng thức giao diện làm việc đỉnh cao của Vaxlou!

## ⌨️ Phím Tắt Tác Vụ Bookmark Bar Toàn Năng

Bạn có thể tự do ghi nhận phím tắt tương tác thông qua **Shortcut Key Recorder** siêu nhạy trong tab Cài đặt (tự động ngăn chặn trôi phím khi gõ văn bản thông thường nhờ *Smart Typing Shield*):

| Tác vụ | Phím tắt mặc định | Mô tả |
| :--- | :--- | :--- |
| **Bật/tắt Panel chính** | `Alt+N` | Đóng/mở nhanh thanh panel Vaxlou bên phải. |
| **Đóng/Mở Sidebar trái** | `Alt+S` | Thu gọn/mở rộng thanh điều hướng native của Gemini. |
| **Thực đơn Bento (⠿)** | `Alt+B` | Mở nhanh menu Bento ở góc trái Bookmark Bar. |
| **Thêm chat vào file** | `Alt+A` | Mở menu gán nhanh cuộc hội thoại hiện tại vào thư mục. |
| **Thao tác trò chuyện** | `Alt+O` | Kích hoạt menu thao tác native (Đổi tên/Xoá/Chia sẻ) của Gemini. |
